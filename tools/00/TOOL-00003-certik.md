# CertiK

**Category:** security-audit
**Access:** web (audit reports)
**Default Evidence Tier when cited as sole source:** 2

Named technical auditor performing direct code/security examination. A single CertiK finding lands at Tier 2 (Independent Confirmation) because it's direct artifact examination, not interpretation of someone else's interpretation.

## Usage in this framework

Referenced in [Source Priority Order](/rules/RULE-001-source-priority-order.md), [Evidence Tier Framework](/frameworks/FRAMEWORK-001-evidence-tier-framework.md), and (for on-chain tools) [On-Chain Verification Framework](/frameworks/FRAMEWORK-002-onchain-verification.md). A per-claim Evidence Tier is always assigned by what was verified this time — a good track record here doesn't upgrade an individual claim's tier; see the corresponding entry, if any, in `/sources/`.
