# Examples

**What belongs here:** worked, illustrative applications of a Rule, Framework, or Playbook — always clearly fictional or clearly marked as illustrative, never presented as real findings. An Example exists to teach a human contributor or ground an AI agent's few-shot context, not to serve as evidence about any real market event.

**Why this category exists:** a Framework states a reasoning model in the abstract (e.g. the Weakest-Link Rule); an Example shows it resolving a concrete case end-to-end. This is the category most directly useful as retrieval context injected alongside a Rule/Framework when an agent needs to *apply* a standard, not just read its definition.
